density_matrix_classes.nlevel.rho_dot(Hamiltonian, Gamma, rho, closed)[source]¶
Calculate the time derivative of the density matrix using the quantum master equation (Steck eq. 5.177 & 5.178 pg 181)
rho_dot = -i / hbar * [H, rho] + Gamma * D[c] * rho
where
D[c] * rho = 1 / 2 * {c_dagger * c, rho}.

Gamma is the decay rate and c are the transition matrices of the n level system. In practice c will be a sum of multiple transition matrices. The documentation also has some extra information. For now only radiative decay is considered (no dephasing).

Parameters:
  • Hamiltonian – Complex array. Hamiltonian.
  • Gamma – Array. Decay matrix.
  • rho – Complex array. density matrix.
Returns:

Complex array. Time derivative of the density matrix.
